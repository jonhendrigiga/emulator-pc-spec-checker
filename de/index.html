FAQ

Genshin ruckelt im Emulator?

DirectX + 6 Kerne + 8GB + mittlere/niedrige Grafik + 60fps + dedizierte GPU erzwingen; auf Throttling achten.

Keine 120 fps?

Nötig: Spiel unterstützt hohe FPS + Emulator auf 120 + Monitor >=120Hz, alle drei.
