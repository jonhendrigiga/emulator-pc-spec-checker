FAQ

Genshin lags in the emulator?

DirectX + 6 cores + 8G + medium-low graphics + 60fps lock + force discrete GPU; watch thermal throttling.

Can't hit 120fps?

Needs game high-fps support + emulator at 120 + monitor >=120Hz, all three.

Will an old PC run it?

LDPlayer's slim core can use ~800MB RAM; 4-8G can run 2-3 instances; VT + acceleration adds ~40%.

Game incompatible / purple-black textures?

Enable ASTC textures; switch Android version (Star Rail/TFT need 64-bit).
