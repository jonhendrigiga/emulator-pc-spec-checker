120 fps - trois conditions

  • Le jeu prend en charge la haute fps
  • Émulateur réglé à 120 FPS
  • Écran >=120 Hz
  • Dans le panneau NVIDIA, désactivez « FPS max », activez la V-Sync

Bases indispensables

  • Activer le VT (+50-100%)
  • Mettre dnplayer.exe en hautes performances sur PC portable bi-GPU
  • Installer dans un chemin ASCII
  • Désactiver les graphismes avancés si VRAM <4 Go

FAQ

Genshin rame sur l'émulateur ?

DirectX + 6 cœurs + 8 Go + graphismes moyens/bas + 60fps + GPU dédié forcé ; attention à la surchauffe.

Pas de 120 fps ?

Il faut jeu compatible + émulateur à 120 + écran >=120 Hz, les trois.

Un vieux PC peut-il tourner ?

Le cœur léger de LDPlayer peut utiliser ~800 Mo ; 4-8 Go font 2-3 instances ; VT + accélération +40%.

Jeu incompatible / textures noir-violet ?

Activez les textures ASTC ; changez la version Android (Star Rail/TFT en 64 bits).
